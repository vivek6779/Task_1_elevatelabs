=>A simple personal profile webpage built using semantic HTML to demonstrate clean structure, accessibility, and best practices in modern web development.

Technologies Used
HTML5
VS Code (or any text editor)
Web Browser (Chrome, Edge, Firefox)

Features
Semantic elements: header, nav, section, article, footer
Profile image with accessible alt text
Internal navigation using anchor links
Structured content for bio, education, skills, and contact info
Well-commented, readable HTML code

Purpose
To writing semantic, accessible HTML without relying on frameworks or snippets.
